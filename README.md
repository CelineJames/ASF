# Achievers Scholars Forum (ASF)

A free community dedicated to helping graduate students put their best foot forward, from crafting outstanding applications to securing scholarships, funding, and academic opportunities worldwide.

## About

Achievers Scholars Forum (ASF) is a platform built for ambitious students who are serious about their academic journey. We provide expert document review, curated resources, and a thriving peer community to help you stand out in competitive scholarship and graduate school applications.

Whether you're working on a personal statement, research proposal, CV, or funding application ASF connects you with the tools and people you need to succeed.

## What We Offer

- **Document Review** — Submit your applications, essays, and academic documents for expert feedback
- **Free Resources** — Access templates, checklists, and guides to strengthen your submissions
- **Community** — Connect with fellow scholars, share experiences, and grow together
- **Scholarship & Funding Guidance** — Stay informed on opportunities and how to apply effectively

## Tech Stack

- [Vite](https://vitejs.dev/) — Build tool
- [React](https://react.dev/) — UI framework
- [TypeScript](https://www.typescriptlang.org/) — Type safety
- [React Hook Form](https://react-hook-form.com/) — Form validation
- CSS Modules — Scoped component styling

## Getting Started

```bash
# Clone the repository
git clone https://github.com/CelineJames/ASF.git

# Navigate into the project
cd ASF

# Install dependencies
npm install

# Start the development server
npm run dev
```

## Project Structure

```
src/
├── assets/          # Images and icons
├── components/      # All page sections
├── data/            # Static data (institutions list, etc.)
├── styles/          # Global CSS variables and tokens
└── main.tsx         # App entry point
```
